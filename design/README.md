# Design Docs

This directory contains documents on design decisions
ordered chronologically, following the naming convention:

```
<decision-date>-<title>.md
```

For working-in-progress documents, use `wip` in place of the decision date.

Once a decision has been made, the document shall not be modified.
To revise a previous decision, create a modified version of the document
with "revision" in the title, and highlight the major changes made.

Not all design decisions needs to be recorded.
